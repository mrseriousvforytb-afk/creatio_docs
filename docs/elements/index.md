# Элементы UI

Справочник по элементам Freedom UI. Каждый элемент — отдельная страница с полным описанием свойств, примерами и подводными камнями.

---

## Поля ввода

| Элемент | Тип | Описание |
|---------|-----|----------|
| [crt.Input](input.md) | Текстовое поле | Однострочный и многострочный ввод текста |
| [crt.NumberInput](numberinput.md) | Числовое поле | Целые и дробные числа, min/max, precision |
| [crt.ComboBox](combobox.md) | Выпадающий список | Lookup-поля, привязка к справочникам |
| [crt.DateTimePicker](datetimepicker.md) | Дата/время | Выбор даты, времени, datetime |
| [crt.Checkbox](checkbox.md) | Чекбокс | Boolean-поля |
